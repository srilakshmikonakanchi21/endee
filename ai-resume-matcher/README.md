## AI Resume Job Matcher using Endee

### Overview
This project matches resumes with job descriptions using semantic search.

### Features
- Upload resume (PDF)
- Extract text
- Convert to embeddings
- Retrieve similar jobs

### Use of Endee
Endee is used as a vector database to:
- Store job embeddings
- Perform similarity search
- Retrieve top job matches

### Run Project
streamlit run ai-resume-matcher/app.py
